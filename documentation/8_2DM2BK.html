Module 2DM2BK

This module converts SMS 2D Mesh Files (*.2dm) to 2D T3 Scalar Meshes (*.t3s) that can be red by BlueKenue. In addition, data files (*.dat) can be converted to a 2D T3 Scalar Mesh (*.t3s). Node strings are converted to 2D Line Sets (*.i2s), boundary nodes for pressure flow to Point Sets (*.xyz).

Example 8: Converting SMS file

A SMS file should be converted to files that can be processed by BlueKenue.

Load the *.2dm file and the *.dat file by use of the according lines. Check the boxes for the desired results and give names for the results files.

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.
