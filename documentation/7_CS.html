Module CS

This module creates a formatted data file from the sections input and output files, that can further be processed by a spreadsheet software. The sections can also be extracted to a dxf file, where the names of the sections and the minimum and maximum flows are displayed. The decimal points for the time steps and the flow can be varied. The size factor influences the symbol and text size in the dxf file. A flow prefix and suffix can be specified.

Example 7: Processing section results

The section results from the simulation case A from the Telemac 2d validation test case donau should be converted to a formatted text file and the sections should be converted to a dxf file.

Select the Open the results file from the test case A with BlueKenue and save the last step from the water depths to a 2D T3 Scalar Mesh (ASCII SingleFrame) (*.t3s). Extract an isoline from the water depth mesh and specify the isolevel value to 0.0 and save it to a 2D Line Set (*.i2s). Load the mesh and the line set into the corresponding lines and define the decimal points for the time steps and the flow, the size factor for the dxf symbols and the flow pre- and suffix.

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.

The resulting formatted data file can now be processed by a spreadsheet software. The obtained DXF file can now be imported to a CAD software. The following picture shows a part from the domain with the sections. The mesh in the background can be obtained by module BK2DXF or LandXML
