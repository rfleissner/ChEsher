ChEsher - BK2DXF

Module BK2DXF

This module converts 2D T3 Scalar Meshes (*.t3s) and 2D or 3D Line Sets (*.i2s, *.i3s) from Blue Kenue™ to a DXF file. The type of the mesh elements can be selected between the types

  • 3D Face and
  • polyline.

Example 2: Converting geometries from Blue Kenue™ to DXF file

The water depths and the 1 meter contour line of the simulation case A of the Telemac 2d validation test case donau should be converted to dxf file format.

Open the results file of the test case A with Blue Kenue™ and save the last step of water depths to a 2D T3 Scalar Mesh (ASCII SingleFrame) (*.t3s). Extract an isoline of the water depth mesh and specify the isolevel value to 1.0 and save it to a 2D Line Set (*.i2s). Go to ChEsher module BK2DXF and load the mesh and the line set into the corresponding lines. Select the type of mesh to be generated. Define the output file names.

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.

The resulting DXF file can now be imported to a CAD software. The following picture shows a part of the domain with the mesh and the contour.
