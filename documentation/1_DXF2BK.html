Module DXF2BK

This module transforms objects from a DXF file to Blue Kenue files. The geometric entities can be from type

  • point,
  • line,
  • 2d polyline and
  • 3d polyline.
They can be converted to
  • Data Point Sets (*.xyz),
  • 2D Line Sets (*.i2s) or
  • 3D Line Sets (*.i3s),
which can be red by Blue Kenue. The geometric entities to be transformed can be selected by DXF layers.

Example 1: Converting DXF entities

Different geometric entities from a given DXF file should be transformed to Blue Kenue formats.

After starting ChEsher with the module DXF2BK the DXF file can be imported through the therefor designated line. By pressing the button add rows in the table are displayed. The button refresh updates the rows with the layers of the DXF file. This also works, if the DXF file was changed meanwhile. Each row stands for a layer from the DXF file, where a file name for each layer has to be entered.

By pressing the Let's go! button the transformation is started. When the progress is done, some information about the converted geometric entities is given.
