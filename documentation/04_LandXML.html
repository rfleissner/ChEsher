ChEsher - LandXML

Module LandXML

With this module a 2D T3 Scalar Mesh (*.t3s) (e.g. bottom, water depth, water surface, etc.) can be converted to a TIN surface of the LandXML file format, which then can be loaded into a civil engineering software. There, the results can be evaluated and processed, flood area maps can be created or cross sections of the river and the foreland can be generated. The module offers a more flexible way for evaluating and processing results of a simulation.

About LandXML

LandXML is a specialized XML data file format containing civil engineering and survey measurement data. The main goals are open design and survey data exchange. For more information visit www.landxml.org.

Example 4: Creating LandXML file

The bottom and the water surface of the Telemac 2d validation test case donau should be converted to a LandXML file. Open the results file of the validation test case donau (case A) with Blue Kenue™ and save the bottom and the last step of the water surface to a 2D T3 Scalar Mesh (ASCII SingleFrame) (*.t3s). Just select the input file, give the surface a name and choose an output filename.

By pressing the Let's go! button the transformation is started. When the process is done, some information about the results is given.
