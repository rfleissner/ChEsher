Module VectorDXF

This module extracts vectors on a given grid from a 2D T3 Vector Mesh (*.t3v) to a DXF file. At first, the U and V components of the 2D T3 Vector Mesh have to be saved as 2D T3 Scalar Mesh (*.t3s). For this the calculator tool from Blue Kenue can be used. The grid spacing can be selected for both x- and y-direction. A bound can be defined to just extract values in the given range.

Example 6: Create velocity field

The velocity field from the Telemac 2d validation test case donau (case A) should be generated.

Open the results file from the test case with Blue Kenue and save the last step from the velocity components U and V to a 2D T3 Scalar Mesh (ASCII SingleFrame) (*.t3s) by using the calculator tool. Load the U and V meshes by use of the according lines. Assign grid spacing values and minimum and maximum values to be extracted, set a vector scale factor and give a name for the output file.

Start the process by pressing the Let's go! button. When the progress is done, some information about the results is given.
