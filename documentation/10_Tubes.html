ChEsher - Tubes

Module Tubes

This module creates a tubes data file out of a 2D T3 Scalar Mesh (*.t3s) (bottom elevation) and a 2D Line Set (*.i2s), whose lines represent tubes. The tubes data file can be used as input for construction work modelling in a Telemac simulation. The tubes are defined in flow direction by node IDs. The module takes the nearest nodes to the line end points, so it is not necessary to define exact coordinates of tubes start- and endpoint. The levels of the tapers are carried over from the elevation of the mesh nodes. Default values for relaxation, head loss coefficients, width and height of the construction work and an indicator specifying a check valve behaviour can be applied.

Example 10: Creating tubes data file

The bottom and the water surface of the Telemac 2d validation test case donau should be converted to a LandXML file. Open the results file of the validation test case donau (case A) with Blue Kenue™ and save the bottom and the last step of the water surface to a 2D T3 Scalar Mesh (ASCII SingleFrame) (*.t3s). Just select the input file, give the surface a name and choose an output filename.

By pressing the Let's go! button the transformation is started. When the process is done, some information about the results is given.
