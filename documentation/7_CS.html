ChEsher - CS

Module CS

This module creates a formatted data file of the sections input and output files, that can further be processed by a spreadsheet software. The sections can also be extracted to a dxf file, where the names of the sections and the minimum and maximum flows are displayed. The decimal points for the time steps and the flow can be varied. The size factor influences the symbol and text size in the dxf file. A flow prefix and suffix can be specified.

Example 7: Processing section results

The section results of the simulation case A of the Telemac 2d validation test case donau should be converted to a formatted text file and the sections should be converted to a dxf file.

Load the files by use of the according lines. Check the boxes for the desired results and give names for the results files.

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.

The resulting formatted data file can now be processed by a spreadsheet software.

The obtained DXF file can now be imported to a CAD software. The following picture shows a part of the domain with the cross sections. The mesh in the background can be obtained by module BK2DXF or LandXML.
