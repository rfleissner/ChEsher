Module BK2DXF

This module converts meshes (2D T3 Scalar Mesh) and line sets (2D and 3D Line Set) from BlueKenue to a dxf file. The type of the mesh can be selected between

  • 3D Face and
  • polyline.

Example 2: Converting BK geometries to DXF

The water depths and the zero depth contour line from the simulation case A from the Telemac 2d validation test case donau should be converted to dxf file format.

Open the results file from the test case A with BlueKenue and save the last step from the water depths to a 2D T3 Scalar Mesh (ASCII SingleFrame) (*.t3s). Extract an isoline from the water depth mesh and specify the isolevel value to 0.0 and save it to a 2D Line Set (*.i2s). Load the mesh and the line set into the corresponding lines and select the type of mesh to be generated. Define the output file names.

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.
