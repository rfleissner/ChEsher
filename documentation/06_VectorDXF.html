ChEsher - VectorDXF

Module VectorDXF

Velocity vectors are given in the nodes of an irregular triangulated mesh. To provide a much better visual interpretation of the velocity field, vectors of a 2D T3 Vector Mesh (*.t3v) are interpolated to a rasterized grid, what can be done with this module. The vector vield is written to a DXF file. The grid spacing can be selected for both x- and y-direction. A bound can be defined to just extract values in the given range.

Example 6: Create velocity field

The velocity field of the Telemac 2d validation test case donau (case A) should be generated.

Open the results file of the test case with Blue Kenue™ and save the last step of the velocity to a 2D T3 Vector Mesh (ASCII SingleFrame) (*.t3v). Load the mesh by use of the according lines. Assign grid spacing values and minimum and maximum values to be extracted, set a vector scale factor and give a name for the output file.

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.

The resulting DXF file can now be imported to a CAD software. The following picture shows a part of the domain with the grid vectors. The mesh in the background comes from a LandXML import after using module LandXML.
