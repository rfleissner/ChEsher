ChEsher - Cont2DXF

Module Cont2DXF

This module creates contour plots from 2D T3 Scalar Meshes (*.t3s) that can be saved as DXF file format. The contours are created for level ranges with different colours. Pre-defined levels can be applied for water depth, water surface difference, flow velocity and bottom shear stress. The levels can be loaded and saved as EnSim ColourScale File (*.cs1), which is used by Blue Kenue™.
The contour plots can be saved as filled contours or as isolines to a DXF file. A name for the DXF layer can be defined. If the option create legend is checked, a legend is created in the right bottom corner of the contour plot. Therefor, a title and a subtitle can be defined for the legend .

Example 9: Creating a contour plot

A wather depth contour plot of the Telemac 2d validation test case donau (case A) should be saved to a DXF file by using the default levels for water depths.

Load the *.t3s file by use of the according line. By pressing set default the default levels for water depths are applied. Define a DXF layer name and insert title and subtitle for the legend. Check the boxes create legend and define some output file names for the contour plot . ffffffffffffff

Start the process by pressing the Let's go! button. When the process is done, some information about the results is given.
